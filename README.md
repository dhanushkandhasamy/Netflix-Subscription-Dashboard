# 🎬 Netflix Subscription Dashboard (Power BI)

## 📊 Project Overview
This project is a **Power BI Data Visualization Dashboard** that analyzes Netflix subscription data, user behavior, and revenue insights.

The dashboard provides a clear understanding of:
- Subscription growth trends
- Revenue distribution
- User demographics
- Plan usage patterns

## 🚀 Features
📈 Interactive dashboard
💰 Revenue analysis
👥 User segmentation
🌍 Region-based insights
📅 Monthly & yearly trends

## 🛠 Tools & Technologies
- Power BI
- Data Cleaning (Excel / CSV)
- Data Visualization

## 📁 Dataset
Dataset: Netflix Subscription Dataset  
Source: Kaggle  
Link: https://www.kaggle.com/datasets/olagokeblissman/netflix-user-behavior-and-subscription-dataset

## 🎯 Key Insights
- Most users prefer premium plans
- Revenue increased steadily over time
- High user concentration in specific regions
